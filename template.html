This is going to make a plot!First I generate a dataframe, then use ‘ggplot’ to make a scatterplot. ##Every single time you need a document, it start from a scratch, you need to define the library, you can put library in the beginning.

plot_df = 
  tibble(
    x = rnorm(n = 1000), # x is normal distribution
    y = 1 + 2 * x + rnorm(n = 1000)
  )

ggplot(plot_df, aes(x = x, y = y)) + geom_point()

ggplot(plot_df, aes(x = x)) + geom_histogram()
